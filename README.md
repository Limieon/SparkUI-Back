SparkUI is my self-created StableDiffusion WebUI. My goal is to create a powerful WebUI that is also visually nice to look at.

SparkUI is highly inspired by [Automatic1111](https://github.com/AUTOMATIC1111/stable-diffusion-webui), [InvokeAI](https://github.com/invoke-ai/InvokeAI) and [ComfyUI](https://github.com/comfyanonymous/ComfyUI).

The project is in high development and also currently only a frontend. No image generation features are currently implemented, as I first want to finish the main parts of the ui.

## Techstack
I am currently using [Uvicorn](https://www.uvicorn.org/) and [FastAPI](https://fastapi.tiangolo.com/) for the web interface.  
[Prisma](https://www.prisma.io/) with [Prisma Client Python](https://prisma-client-py.readthedocs.io/en/stable/) is currently used for the database.
